# Ankita-Dasana

Portfolio website for **Ankita Dasana** — Creative Producer, Director & Social Media Manager.

Built with plain **HTML, CSS & JavaScript** (no build step). Bold, trend-aware and fully responsive.

## Highlights
- Animated hero with floating stat badges
- Brand marquee & animated counters
- Horizontal scroll/drag case-study cards with engagement metrics
- Experience timeline & performance-marketing certification
- Scroll progress bar, active-nav highlight, card hover-tilt & magnetic buttons

## Run locally
Just open `index.html` in any browser.

## Structure
```
index.html      # markup
styles.css      # styles & responsive layout
script.js       # interactions & animations
assets/         # photos + certificate
```
